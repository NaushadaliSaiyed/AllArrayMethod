The concat() method concatenates (joins) two or more arrays:

const arr1 = ["Naushad","Neha","Azhar","Shoeb","Sharif","Mahemud","Vikas","Rohit","Sohil","Faizan","Adnan","Akash"];

Examples

1. Copy the first two array elements to the last two array elements:

copyWithin() copies array elements to another position in an array, overwriting existing values:

const arr1 = ["Naushad","Neha","Azhar","Shoeb","Sharif","Mahemud","Vikas","Rohit","Sohil","Faizan","Adnan","Akash"];
arr1.copyWithin(6,10)
OUTPUT

2. Copy the first two array elements to the third and fourth position:

copyWithin() copies array elements to another position in an array, overwriting existing values.

const arr1 = ["Naushad","Neha","Azhar","Shoeb","Sharif","Mahemud","Vikas","Rohit","Sohil","Faizan","Adnan","Akash", "Akram","Akshat","Akshay","Aman","Anamika","Anjali","Anup","Anwar","Arif","Ashok","Bhogendra","Dharmik" ];
arr1.copyWithin(2,0,2)
OUTPUT
